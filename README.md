# Luxurious Transportation

Website for a family-owned Las Vegas party-bus business. The project replaces a planned paid website rebuild with a focused, AI-assisted implementation that can be maintained without ongoing agency support.

## Product Goal

Present the transportation service professionally and turn visitors into customer inquiries. The site should remain a polished, fast, mobile-first landing page with a clear path to contact the business.

## Scope

Keep the product intentionally simple. Add complexity only when it directly improves customer trust, inquiry conversion, accessibility, performance, or maintainability.

## Development

```bash
npm ci
npm run dev
```

Before submitting changes, run:

```bash
npm run lint
npm run build
```
